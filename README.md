RateNest is a full-stack **PERN** application that allows users to share products, write reviews, and rate products with secure authentication.

---

## Tech Stack

### Frontend

* React
* TypeScript
* Tailwind CSS
* DaisyUI
* TanStack Query

### Backend

* Node.js
* Express.js
* Drizzle ORM

### Database

* PostgreSQL

---

## Features

* User authentication (Login / Register)
* Add and share products
* Write product reviews
* Product rating system
* Review pagination
* Responsive UI
* Secure REST API
